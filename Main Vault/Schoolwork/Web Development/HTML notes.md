### Always Include

```
<!DOCTYPE html>
<html lang = "en">
	<head>
		<meta charset = "UTF-8">
		<title>TITLE</title>
		<link rel="stylesheet" href="docname.css">
	</head>
	<body>
	
	</body>
</html>
<script src=""></script>
```




### Grouping
- \<head>
- \<body>
- \<section>
- \<!-- -->
	- Commenting


### **UI**
\<input>
- User input field
- \<input type="text" id="id">

\<button>
- Runs a piece of javascript
- Make sure the code exists in the html bracket before it tries to run it
- \<button tyope="button" onclick="JSHERE">ButtonLabel\</button>

### Elements
\<h1> \<h2> \<h3>
- Headers

\<p>
- Paragraphs
- \<p id = "">

\<img src="" alt="">
- Images
- \<img src="" alt="" height="" width="">

\<a href="">\<a>
- Links

\<br>
- Line Breaks

\<ul> \<ol>
- Unordered list
- Ordered list
- Lists use \<li> for list elements
- \<ol type="">
	- 1 = numbered
	- A = capital alphabetical
	- I = roman numeral
	- a = lowercase alphabetical
- Create and enclose a list element for each sub-list

\<table>
- Table
- \<caption>
- \<tr>
	- Row
	- Like a list of cells
	- \<th>
		- Row headers
	- \<td>
		- Row data
		- \<td rowspan="3">

\<script>
- Encloses JavaScript
- \<script src="docname.js>
- Best practice to put it after all other HTML

\<input>
- Used to allow users to input text or numbers

\<link>
- Not entirely sure
- Used to connect external CSS
- \<link rel="stylesheet" href="docname.css">
