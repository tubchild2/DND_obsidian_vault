TTE Web Version

##### The Goal
Simpler, free, easier, and more creatively flexible version of DNDBeyond
People can make campaigns centered around whatever weird idea they want and share it, then users can make characters with the in-app system. Art style themed around arts and crafts like Little Big Planet.

Later server systems will allow for accounts, content sharing and rating, and active character sheets


##### Basic Version
-- PROOF OF CONCEPT -- 
Make an SQL relational database in JS using sql.js
Create a prototype database for a campaign
Create test content directly from the code

-- CONTENT CREATOR --
Create new / import database
Assemble content in selected database
Export database

-- CHARACTER CREATOR --
Import a database
Assemble character
Export character sheet
